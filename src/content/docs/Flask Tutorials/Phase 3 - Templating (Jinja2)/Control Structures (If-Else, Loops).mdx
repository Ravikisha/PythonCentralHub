Control structures are how you build dynamic pages.

## If / elif / else

```html
{% if user %}
  <p>Hello, {{ user.username }}!</p>
{% elif guest_name %}
  <p>Hello, {{ guest_name }}!</p>
{% else %}
  <p>Hello, stranger!</p>
{% endif %}
```

## For loops

```html
<ul>
  {% for item in items %}
    <li>{{ item }}</li>
  {% endfor %}
</ul>
```

### Loop helpers

Jinja provides `loop` inside loops:

- `loop.index` (1-based)
- `loop.index0` (0-based)
- `loop.first`, `loop.last`

```html
{% for user in users %}
  <p>{{ loop.index }}. {{ user.username }}</p>
{% endfor %}
```

## Handling empty lists

```html
{% if users %}
  <!-- show list -->
{% else %}
  <p>No users found.</p>
{% endif %}
```

## Keep logic light

Prefer to compute complex values in Python and pass them in.

Templates should be readable.
