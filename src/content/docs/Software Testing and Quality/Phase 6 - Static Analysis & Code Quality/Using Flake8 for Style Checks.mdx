## What Flake8 checks

- style (PEP8-ish)
- unused imports
- simple logical issues

## Run

```bash
flake8 .
```

## Example configuration

In `.flake8`:

```ini
[flake8]
max-line-length = 88
extend-ignore = E203,W503
exclude = .git,__pycache__,.venv,build,dist
```

## Plugins (popular)

- `flake8-bugbear`
- `flake8-comprehensions`
- `flake8-docstrings`

## Tip

If you use Black, align flake8 ignores with Black defaults.
