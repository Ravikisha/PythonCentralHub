## What Radon measures

- **Cyclomatic complexity** (branches/paths)
- **Maintainability index**

High complexity often means:

- harder to test
- more bugs
- slower changes

## Run

```bash
radon cc -a your_package
```

Maintainability:

```bash
radon mi your_package
```

## How to use results

- identify hotspots
- refactor into smaller functions
- add tests around risky logic first
