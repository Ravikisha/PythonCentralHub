## Why HTML reports

Useful for:

- sharing results with teams
- CI artifacts
- historical tracking

## Common plugin

- `pytest-html`

## Example usage

```bash
pytest --html=report.html --self-contained-html
```

## Tips

- store reports as CI artifacts
- include environment info and versions
