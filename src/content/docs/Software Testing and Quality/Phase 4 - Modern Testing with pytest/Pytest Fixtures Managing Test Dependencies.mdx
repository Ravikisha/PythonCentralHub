## What is a fixture?

A fixture is a reusable function that provides a value to tests.

Use fixtures for:

- test data factories
- temporary resources
- dependency setup (clients, config)

## Basic fixture

```python title="test_fixture_basic.py" showLineNumbers{1}
import pytest


@pytest.fixture
def user():
    return {"name": "Alice", "role": "admin"}


def test_user_role(user):
    assert user["role"] == "admin"
```

## Fixture scope

You can control how often fixtures run:

- `function` (default)
- `class`
- `module`
- `session`

```python title="fixture_scope.py" showLineNumbers{1}
import pytest


@pytest.fixture(scope="module")
def expensive_resource():
    return [1, 2, 3]
```

## Yield fixtures (cleanup)

```python title="fixture_yield.py" showLineNumbers{1}
import pytest


@pytest.fixture
def temp_config_file(tmp_path):
    p = tmp_path / "config.json"
    p.write_text('{"mode": "test"}')
    yield p
    # cleanup happens automatically for tmp_path
```
