## What you’ll learn

This module covers both:

- **QA theory** (how teams ensure quality)
- **hands-on testing in Python** (unittest, pytest, automation)

## Why this matters

Testing is how engineers ship changes confidently.

Good testing practices reduce:

- production bugs
- incident time
- expensive rework

Start with **Phase 1** to understand the fundamentals, then move into unit testing and automation.
