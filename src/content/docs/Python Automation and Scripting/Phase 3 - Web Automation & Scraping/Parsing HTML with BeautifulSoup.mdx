## Install

Common package name: `beautifulsoup4`

## Parse HTML

```python title="bs4_parse.py" showLineNumbers{1}
from bs4 import BeautifulSoup

html = """
<html><body>
  <h1 class='title'>Hello</h1>
  <a href='https://example.com'>Link</a>
</body></html>
"""

soup = BeautifulSoup(html, "html.parser")
print(soup.h1.get_text(strip=True))
print(soup.a["href"])
```

## Find elements

```python title="bs4_find.py" showLineNumbers{1}
from bs4 import BeautifulSoup

soup = BeautifulSoup("<p class='x'>A</p><p class='y'>B</p>", "html.parser")

print(soup.find("p", class_="y").text)
print([p.text for p in soup.find_all("p")])
```

## Use CSS selectors

```python title="bs4_select.py" showLineNumbers{1}
from bs4 import BeautifulSoup

soup = BeautifulSoup("<div><span class='price'>$10</span></div>", "html.parser")
print(soup.select_one("span.price").text)
```
