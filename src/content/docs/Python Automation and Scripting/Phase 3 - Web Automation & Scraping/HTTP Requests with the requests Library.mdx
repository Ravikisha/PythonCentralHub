## Always use timeouts

```python title="requests_get.py" showLineNumbers{1}
import requests

r = requests.get("https://httpbin.org/get", timeout=10)
r.raise_for_status()
print(r.status_code)
print(r.json())
```

## Headers and User-Agent

```python title="requests_headers.py" showLineNumbers{1}
import requests

headers = {"User-Agent": "PythonCentralHubBot/1.0"}
r = requests.get("https://httpbin.org/headers", headers=headers, timeout=10)
print(r.json())
```

## Sessions (cookies + connection pooling)

```python title="requests_session.py" showLineNumbers{1}
import requests

with requests.Session() as s:
    s.headers.update({"User-Agent": "MyBot/1.0"})
    r = s.get("https://httpbin.org/cookies/set?hello=world", timeout=10)
    r.raise_for_status()

    r2 = s.get("https://httpbin.org/cookies", timeout=10)
    print(r2.json())
```
