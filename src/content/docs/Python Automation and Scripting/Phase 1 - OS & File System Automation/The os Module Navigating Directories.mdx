## What `os` is for

The `os` module provides:

- current working directory
- directory listing
- environment variables
- basic path manipulation (but prefer `pathlib` for paths)

## Current working directory

```python title="cwd.py" showLineNumbers{1}
import os

print(os.getcwd())
```

## List files

```python title="listdir.py" showLineNumbers{1}
import os

for name in os.listdir("."):
    print(name)
```

## Change directory

```python title="chdir.py" showLineNumbers{1}
import os

os.chdir("/tmp")
print(os.getcwd())
```

## Environment variables

```python title="env_vars.py" showLineNumbers{1}
import os

print(os.environ.get("HOME"))
```

## Notes

For paths, use `pathlib.Path` whenever possible.
