## What `shutil` is for

`shutil` (shell utilities) helps with:

- copying files/folders
- moving files
- removing directory trees

## Copying files

```python title="copy_file.py" showLineNumbers{1}
import shutil

shutil.copy2("source.txt", "backup_source.txt")
```

`copy2` attempts to preserve metadata.

## Copying folders

```python title="copy_tree.py" showLineNumbers{1}
import shutil

shutil.copytree("my_folder", "my_folder_backup", dirs_exist_ok=True)
```

## Moving files

```python title="move_file.py" showLineNumbers{1}
import shutil

shutil.move("old/name.txt", "new/name.txt")
```

## Deleting folders (danger!)

```python title="remove_tree.py" showLineNumbers{1}
import shutil

shutil.rmtree("danger_folder")
```

Always do a dry run before destructive operations.
