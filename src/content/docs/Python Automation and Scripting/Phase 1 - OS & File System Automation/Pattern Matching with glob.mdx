## What glob does

`glob` finds files using wildcard patterns:

- `*.txt`
- `data_??.csv`
- recursive patterns with `**`

## Using `glob` module

```python title="glob_basic.py" showLineNumbers{1}
from glob import glob

for path in glob("*.py"):
    print(path)
```

## Recursive glob

```python title="glob_recursive.py" showLineNumbers{1}
from glob import glob

for path in glob("**/*.md", recursive=True):
    print(path)
```
