## Type text

```python title="type_text.py" showLineNumbers{1}
import pyautogui

pyautogui.write("hello", interval=0.05)
pyautogui.press("enter")
```

## Hotkeys

```python title="hotkeys.py" showLineNumbers{1}
import pyautogui

# Example: Ctrl+L (address bar in browsers)
pyautogui.hotkey("ctrl", "l")
```

## Tip

UI automation is fragile; use stable workflows and add delays.
