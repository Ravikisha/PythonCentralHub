## Alert

```python title="alert.py" showLineNumbers{1}
import pyautogui

pyautogui.alert("Automation completed", title="Done")
```

## Confirm

```python title="confirm.py" showLineNumbers{1}
import pyautogui

ans = pyautogui.confirm("Proceed?", buttons=["Yes", "No"])
print(ans)
```
