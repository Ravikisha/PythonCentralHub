## Move and click

```python title="mouse_click.py" showLineNumbers{1}
import pyautogui

pyautogui.moveTo(200, 200, duration=0.5)
pyautogui.click()
```

## Drag

```python title="mouse_drag.py" showLineNumbers{1}
import pyautogui

pyautogui.dragTo(400, 400, duration=0.5, button="left")
```

## Scroll

```python title="scroll.py" showLineNumbers{1}
import pyautogui

pyautogui.scroll(-500)
```
