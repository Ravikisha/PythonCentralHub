## Why schedule

It’s simple for:

- “run every hour”
- “run at 09:00 daily”

## Example

```python title="schedule_example.py" showLineNumbers{1}
import schedule
import time


def job():
    print("running")


schedule.every(10).seconds.do(job)

while True:
    schedule.run_pending()
    time.sleep(1)
```

## Notes

- this keeps a process running
- for server deployment, cron/systemd is usually better
