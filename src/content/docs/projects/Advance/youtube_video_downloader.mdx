2. **Video Extraction and Format Selection Functions**
```python title="youtube_video_downloader.py" showLineNumbers{1}
def download_video(url, resolution='720p'):
    yt = YouTube(url)
    stream = yt.streams.filter(res=resolution, file_extension='mp4').first()
    if stream:
        stream.download()
        return f"Downloaded {yt.title}"
    return "Resolution not available."
```

3. **CLI Interface and Error Handling**
```python title="youtube_video_downloader.py" showLineNumbers{1}
def main():
    print("YouTube Video Downloader")
    while True:
        cmd = input('> ')
        if cmd == 'download':
            url = input("YouTube URL: ")
            resolution = input("Resolution (e.g., 720p): ")
            print(download_video(url, resolution))
        elif cmd == 'exit':
            break
        else:
            print("Unknown command. Type 'download' or 'exit'.")

if __name__ == "__main__":
    main()
```
