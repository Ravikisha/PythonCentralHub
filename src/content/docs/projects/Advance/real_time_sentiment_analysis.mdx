```python title="real_time_sentiment_analysis.py" showLineNumbers{1}
def analyze_sentiment(text):
    # Dummy sentiment analysis (for demo)
    if 'good' in text.lower():
        return 'positive'
    elif 'bad' in text.lower():
        return 'negative'
    return 'neutral'
```

3. **CLI Interface and Error Handling**
```python title="real_time_sentiment_analysis.py" showLineNumbers{1}
def main():
    print("Real-Time Sentiment Analysis")
    while True:
        cmd = input('> ')
        if cmd == 'analyze':
            text = input("Text to analyze: ")
            print(analyze_sentiment(text))
        elif cmd == 'exit':
            break
        else:
            print("Unknown command. Type 'analyze' or 'exit'.")

if __name__ == "__main__":
    main()
```
