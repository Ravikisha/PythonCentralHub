2. **Translation and Text Processing Functions**
```python title="language_translation_app.py" showLineNumbers{1}
def translate_text(text, dest_lang):
    translator = Translator()
    result = translator.translate(text, dest=dest_lang)
    return result.text
```

3. **CLI Interface and Error Handling**
```python title="language_translation_app.py" showLineNumbers{1}
def main():
    print("Language Translation App")
    while True:
        cmd = input('> ')
        if cmd == 'translate':
            text = input("Text to translate: ")
            dest_lang = input("Destination language (e.g., 'es' for Spanish): ")
            try:
                translated = translate_text(text, dest_lang)
                print(f"Translated: {translated}")
            except Exception as e:
                print(f"Error: {e}")
        elif cmd == 'exit':
            break
        else:
            print("Unknown command. Type 'translate' or 'exit'.")

if __name__ == "__main__":
    main()
```
