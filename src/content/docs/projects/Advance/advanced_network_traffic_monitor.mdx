1. **Import Libraries and Setup Capture**

```python title="advanced_network_traffic_monitor.py" showLineNumbers{1}
from scapy.all import sniff
import matplotlib.pyplot as plt
```

2. **Packet Analysis Function**
```python title="advanced_network_traffic_monitor.py" showLineNumbers{1}
protocol_counts = {'TCP': 0, 'UDP': 0, 'ICMP': 0, 'Other': 0}

def analyze_packet(packet):
    if packet.haslayer('TCP'):
        protocol_counts['TCP'] += 1
    elif packet.haslayer('UDP'):
        protocol_counts['UDP'] += 1
    elif packet.haslayer('ICMP'):
        protocol_counts['ICMP'] += 1
    else:
        protocol_counts['Other'] += 1
```

3. **Start Packet Capture**
```python title="advanced_network_traffic_monitor.py" showLineNumbers{1}
def start_capture():
    print("Capturing packets... Press Ctrl+C to stop.")
    try:
        sniff(prn=analyze_packet, store=0)
    except PermissionError:
        print("Error: Run as administrator/root.")
```

4. **Visualize Protocol Distribution**
```python title="advanced_network_traffic_monitor.py" showLineNumbers{1}
def visualize():
    labels = list(protocol_counts.keys())
    sizes = list(protocol_counts.values())
    plt.pie(sizes, labels=labels, autopct='%1.1f%%')
    plt.title('Protocol Distribution')
    plt.show()
```

5. **CLI Interface and Error Handling**
```python title="advanced_network_traffic_monitor.py" showLineNumbers{1}
def main():
    print("Advanced Network Traffic Monitor")
    while True:
        cmd = input('> ')
        if cmd == 'capture':
            start_capture()
        elif cmd == 'visualize':
            visualize()
        elif cmd == 'exit':
            break
        else:
            print("Unknown command. Type 'capture', 'visualize', or 'exit'.")

if __name__ == "__main__":
    main()
```
