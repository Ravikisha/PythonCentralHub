2. **Key Frame Extraction Function**
```python title="ai_powered_video_summarizer.py" showLineNumbers{1}
def extract_key_frames(video_path):
    cap = cv2.VideoCapture(video_path)
    frames = []
    while cap.isOpened():
        ret, frame = cap.read()
        if not ret:
            break
        frames.append(frame)
    cap.release()
    key_frames = [frames[0]] if frames else []
    for i in range(1, len(frames)):
        diff = np.sum(np.abs(frames[i].astype(np.int32) - frames[i-1].astype(np.int32)))
        if diff > 1e6:
            key_frames.append(frames[i])
    return key_frames
```

3. **CLI Interface and Error Handling**
```python title="ai_powered_video_summarizer.py" showLineNumbers{1}
def main():
    print("AI-powered Video Summarizer")
    video_path = input("Enter video path: ")
    try:
        key_frames = extract_key_frames(video_path)
        print(f"Extracted {len(key_frames)} key frames.")
    except Exception as e:
        print(f"Error: {e}")

if __name__ == "__main__":
    main()
```
