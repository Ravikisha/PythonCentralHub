2. **Password Generation Function**
```python title="advanced_password_manager.py" showLineNumbers{1}
def generate_password(length=16):
    alphabet = 'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#$%^&*()'
    return ''.join(secrets.choice(alphabet) for _ in range(length))
```

3. **Encryption and Storage**
```python title="advanced_password_manager.py" showLineNumbers{1}
def save_password(service, password, key):
    f = Fernet(key)
    encrypted = f.encrypt(password.encode())
    with open(f'{service}.pwd', 'wb') as file:
        file.write(encrypted)
```

4. **Password Retrieval**
```python title="advanced_password_manager.py" showLineNumbers{1}
def retrieve_password(service, key):
    f = Fernet(key)
    try:
        with open(f'{service}.pwd', 'rb') as file:
            encrypted = file.read()
        return f.decrypt(encrypted).decode()
    except Exception:
        return None
```

5. **CLI Interface and Error Handling**
```python title="advanced_password_manager.py" showLineNumbers{1}
def main():
    print("Advanced Password Manager")
    key = Fernet.generate_key()
    master = getpass.getpass("Enter master password: ")
    while True:
        cmd = input('> ')
        if cmd == 'generate':
            length = int(input("Password length: "))
            pwd = generate_password(length)
            print(f"Generated Password: {pwd}")
        elif cmd == 'save':
            service = input("Service name: ")
            pwd = getpass.getpass("Password: ")
            save_password(service, pwd, key)
            print("Password saved.")
        elif cmd == 'retrieve':
            service = input("Service name: ")
            pwd = retrieve_password(service, key)
            print(f"Password: {pwd if pwd else 'Not found'}")
        elif cmd == 'exit':
            break
        else:
            print("Unknown command. Type 'generate', 'save', 'retrieve', or 'exit'.")

if __name__ == "__main__":
    main()
```
