## Abstract

Build a GitHub Profile Viewer that fetches and displays GitHub user profile information using the GitHub API. The app provides a GUI for entering usernames and viewing profile details. This project demonstrates API integration, data parsing, and GUI development in Python.

## Prerequisites
- Python 3.6 or above
- Text Editor or IDE
- Basic understanding of Python syntax
- Familiarity with Tkinter for GUI development
- Knowledge of HTTP requests and JSON parsing

## Getting Started
### Creating a new project

2. Create a new file inside the folder and name it `github_profile_viewer.py`.
3. Open the project folder in your favorite text editor or IDE.

## Explanation

### Fetching Profile Data
The app uses the GitHub API to fetch user profile information:
```python title="github_profile_viewer.py" showLineNumbers={1}
url = f"https://api.github.com/users/{username}"
response = requests.get(url)
if response.status_code == 200:
    data = response.json()
```

### Displaying Profile Information
Profile details are shown in a Tkinter Text widget:
```python title="github_profile_viewer.py" showLineNumbers={1}
self.result_text.insert(END, f"Name: {data.get('name', 'N/A')}\n")
self.result_text.insert(END, f"Bio: {data.get('bio', 'N/A')}\n")
self.result_text.insert(END, f"Public Repos: {data.get('public_repos', 'N/A')}\n")
```

## Running the Application
1. Save the file.
2. Install required dependencies:
```bash
pip install requests
```
3. Run the application:
```bash
python github_profile_viewer.py
```
