## Abstract

Build a QR Code Attendance System that generates QR codes for users and scans them to mark attendance. The app provides a GUI for generating and scanning codes. This project demonstrates QR code generation, image processing, and GUI development in Python.

## Prerequisites
- Python 3.6 or above
- Text Editor or IDE
- Basic understanding of Python syntax
- Familiarity with Tkinter for GUI development
- Knowledge of QR code generation and OpenCV

## Explanation

### Generating QR Codes
The app uses the `qrcode` library to generate QR codes:
```python title="qr_code_attendance_system.py" showLineNumbers={1}
qr = qrcode.QRCode(version=1, box_size=10, border=5)
qr.add_data(user_id)
qr.make(fit=True)
img = qr.make_image(fill="black", back_color="white")
img.save(img_path)
```

### Scanning QR Codes
OpenCV is used to scan QR codes and mark attendance:
```python title="qr_code_attendance_system.py" showLineNumbers={1}
cap = cv2.VideoCapture(0)
detector = cv2.QRCodeDetector()
```
