## else in try/except

`else` runs **only if no exception occurred**.

```python title="try_else.py" showLineNumbers{1}
try:
    x = int("42")
except ValueError:
    print("Invalid number")
else:
    print("Parsed successfully:", x)
```

## finally in try/except

`finally` runs **always** (success or error). This is useful for cleanup.

```python title="try_finally.py" showLineNumbers{1}
f = None
try:
    f = open("data.txt", "r")
    print(f.read())
except FileNotFoundError:
    print("File not found")
finally:
    if f is not None:
        f.close()
        print("File closed")
```

## Prefer context managers when possible

Instead of manual `.close()`, use `with`.

```python title="with_open.py" showLineNumbers{1}
try:
    with open("data.txt", "r") as f:
        print(f.read())
except FileNotFoundError:
    print("File not found")
```

## When is finally important?

- closing files
- releasing locks
- closing DB connections
- cleaning up temporary files
