## What is an Event?

An `Event` is a simple flag:

- one thread sets it
- other threads wait until it’s set

## Example

```python title="event_example.py" showLineNumbers{1}
import threading
import time

ready = threading.Event()


def waiter():
    print("Waiting...")
    ready.wait()  # blocks
    print("Go!")


def setter():
    time.sleep(1)
    ready.set()


t1 = threading.Thread(target=waiter)
t2 = threading.Thread(target=setter)

t1.start(); t2.start()
t1.join(); t2.join()
```

## Resetting an event

- `event.clear()` resets it to false.
- Useful for repeated cycles.
