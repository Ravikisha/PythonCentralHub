## Why asyncio.Queue

`asyncio.Queue` enables:

- safe communication between coroutines
- backpressure (limit queue size)

## Example pipeline

```python title="async_queue.py" showLineNumbers{1}
import asyncio


async def producer(q: asyncio.Queue):
    for i in range(10):
        await q.put(i)
        print("produced", i)
    await q.put(None)  # sentinel


async def consumer(q: asyncio.Queue):
    while True:
        item = await q.get()
        try:
            if item is None:
                break
            print("consumed", item)
            await asyncio.sleep(0.1)
        finally:
            q.task_done()


async def main():
    q = asyncio.Queue(maxsize=5)

    p = asyncio.create_task(producer(q))
    c = asyncio.create_task(consumer(q))

    await p
    await q.join()
    await c


asyncio.run(main())
```

## Tips

- Use a sentinel to stop consumers.
- Use `maxsize` for backpressure.
