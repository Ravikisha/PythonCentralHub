## Basic Thread usage

```python title="basic_thread.py" showLineNumbers{1}
import threading


def greet(name: str) -> None:
    print("Hello", name)


t = threading.Thread(target=greet, args=("Ravi",))
t.start()
t.join()
print("Main thread finished")
```

## Naming threads

Naming is helpful for logs.

```python title="thread_name.py" showLineNumbers{1}
import threading


def work():
    print("Running in:", threading.current_thread().name)


t = threading.Thread(target=work, name="worker-1")
t.start()
t.join()
```

## Common mistakes

### 1) Forgetting join()

If you don’t `join()`, the main program may exit early in some cases.

### 2) Sharing data without protection

Threads share memory. You need locks (next pages).
