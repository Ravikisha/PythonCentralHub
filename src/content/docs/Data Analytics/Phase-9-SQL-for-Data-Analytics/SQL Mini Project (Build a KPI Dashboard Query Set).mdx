## Goal

Create a mini “query pack” for a dashboard.

Assume tables:

- `users(user_id, created_at, country, plan)`
- `orders(order_id, user_id, order_ts, amount)`
- `events(user_id, event_ts, event_name)`

## 1) Daily active users (DAU)

```sql title="DAU" showLineNumbers{1}
SELECT
  DATE(event_ts) AS day,
  COUNT(DISTINCT user_id) AS dau
FROM events
GROUP BY DATE(event_ts)
ORDER BY day;
```

## 2) Daily revenue

```sql title="Daily revenue" showLineNumbers{1}
SELECT
  DATE(order_ts) AS day,
  SUM(amount) AS revenue
FROM orders
GROUP BY DATE(order_ts)
ORDER BY day;
```

## 3) Top countries by revenue

```sql title="Top countries" showLineNumbers{1}
SELECT
  u.country,
  SUM(o.amount) AS revenue
FROM orders o
JOIN users u ON u.user_id = o.user_id
GROUP BY u.country
ORDER BY revenue DESC
LIMIT 10;
```

## 4) ARPU (avg revenue per user)

```sql title="ARPU" showLineNumbers{1}
WITH revenue_by_user AS (
  SELECT user_id, SUM(amount) AS revenue
  FROM orders
  GROUP BY user_id
)
SELECT
  AVG(revenue) AS arpu
FROM revenue_by_user;
```

## Deliverable

- 5–10 queries you can reuse
- Notes about assumptions (time zone, filters, returns)
