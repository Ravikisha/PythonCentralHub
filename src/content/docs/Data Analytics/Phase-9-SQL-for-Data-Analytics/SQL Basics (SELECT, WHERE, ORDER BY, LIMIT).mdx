## SELECT

Choose the columns you need.

```sql title="Select columns" showLineNumbers{1}
SELECT user_id, country, created_at
FROM users;
```

## WHERE

Filter rows.

```sql title="Filter by country" showLineNumbers{1}
SELECT user_id, country
FROM users
WHERE country = 'IN';
```

### Common operators

- `=`, `!=`, `>`, `<`, `>=`, `<=`
- `IN (...)`
- `BETWEEN a AND b`
- `LIKE '%text%'`
- `IS NULL` / `IS NOT NULL`

```sql title="Multiple filters" showLineNumbers{1}
SELECT user_id, created_at, plan
FROM users
WHERE country IN ('IN', 'US')
  AND plan != 'free'
  AND created_at >= '2025-01-01';
```

## ORDER BY

```sql title="Sort newest users" showLineNumbers{1}
SELECT user_id, created_at
FROM users
ORDER BY created_at DESC
LIMIT 20;
```

## LIMIT

Limits are great for exploration.

```sql title="Inspect sample rows" showLineNumbers{1}
SELECT *
FROM orders
LIMIT 50;
```
