## Why linear algebra is useful

Linear algebra appears in:

- Regression
- Optimization
- Dimensionality reduction
- Correlation and covariance

NumPy provides `numpy.linalg` for common operations.

## Dot product

```python title="dot" showLineNumbers{1}
import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print(np.dot(a, b))  # 1*4 + 2*5 + 3*6
```

## Matrix multiplication

```python title="matmul" showLineNumbers{1}
import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[10, 20], [30, 40]])

print(A @ B)
print(np.matmul(A, B))
```

## Transpose

```python title="transpose" showLineNumbers{1}
import numpy as np

A = np.array([[1, 2], [3, 4]])
print(A.T)
```

## Determinant and inverse

```python title="det-inv" showLineNumbers{1}
import numpy as np

A = np.array([[1, 2], [3, 4]])

det = np.linalg.det(A)
inv = np.linalg.inv(A)

print("det:", det)
print("inv:\n", inv)
```

:::caution
Inverses can be numerically unstable. Often you should solve systems directly (`np.linalg.solve`).
:::

## Solve a system of equations

Solve `A x = b`:

```python title="solve" showLineNumbers{1}
import numpy as np

A = np.array([[2, 1], [1, 3]])
b = np.array([8, 13])

x = np.linalg.solve(A, b)
print(x)
```

## Eigenvalues and eigenvectors

```python title="eig" showLineNumbers{1}
import numpy as np

A = np.array([[2, 0], [0, 3]])
vals, vecs = np.linalg.eig(A)

print("eigenvalues:", vals)
print("eigenvectors:\n", vecs)
```

## Norms (vector length)

```python title="norm" showLineNumbers{1}
import numpy as np

v = np.array([3, 4])
print(np.linalg.norm(v))  # 5
```
