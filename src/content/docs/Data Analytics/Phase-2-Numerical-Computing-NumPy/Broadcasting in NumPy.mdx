## What is broadcasting?

**Broadcasting** is NumPy’s ability to perform operations on arrays with different shapes by automatically expanding (virtually) smaller arrays.

This is a major reason NumPy code is concise and fast.

## Simple example: add scalar

```python title="scalar" showLineNumbers{1}
import numpy as np

arr = np.array([1, 2, 3])
print(arr + 10)  # [11 12 13]
```

Here, the scalar `10` is broadcast to match the shape `(3,)`.

## Example: add vector to matrix

```python title="vector-matrix" showLineNumbers{1}
import numpy as np

mat = np.array([
    [1, 2, 3],
    [4, 5, 6]
])
vec = np.array([10, 20, 30])

print(mat + vec)
```

`vec` (shape `(3,)`) broadcasts across each row.

## Broadcasting rules (must know)

When operating on two arrays, NumPy compares shapes from the **trailing dimension**.

Two dimensions are compatible when:

1. They are equal, OR
2. One of them is `1`

If dimensions are incompatible → broadcasting error.

## Example: column vector + matrix

```python title="column" showLineNumbers{1}
import numpy as np

mat = np.array([
    [1, 2, 3],
    [4, 5, 6]
])
col = np.array([100, 200]).reshape(2, 1)

print(mat + col)
```

`col` has shape `(2, 1)` and broadcasts across columns.

## Common broadcasting error

```python title="error" showLineNumbers{1}
import numpy as np

mat = np.zeros((2, 3))
vec = np.array([1, 2])

# mat + vec -> ValueError (shapes (2,3) and (2,) not compatible)
```

Fix by reshaping `vec` to a column vector if that’s what you intend:

```python title="fix" showLineNumbers{1}
vec = vec.reshape(2, 1)
print(mat + vec)
```

## Why broadcasting is useful in analytics

- Normalize columns: `X / X.max(axis=0)`
- Center data: `X - X.mean(axis=0)`
- Apply weights: `X * weights`
