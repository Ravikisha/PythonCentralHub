## Understanding shape

`shape` tells the size of each dimension.

```python title="shape" showLineNumbers{1}
import numpy as np

arr = np.array([[1, 2, 3], [4, 5, 6]])
print(arr.shape)  # (2, 3)
```

## Reshape with `.reshape()`

Reshape changes the dimension layout without changing the data.

```python title="reshape" showLineNumbers{1}
import numpy as np

arr = np.arange(1, 13)  # 1..12
mat = arr.reshape(3, 4) # 3 rows, 4 cols
print(mat)
```

### Using `-1` (auto infer)

```python title="reshape-auto" showLineNumbers{1}
import numpy as np

arr = np.arange(12)
mat = arr.reshape(3, -1)
print(mat.shape)  # (3, 4)
```

:::caution
The total size must match.
If the array has 12 elements, only shapes that multiply to 12 are valid.
:::

## Flattening arrays

### `.ravel()` (often a view)

```python title="ravel" showLineNumbers{1}
import numpy as np

mat = np.array([[1, 2], [3, 4]])
flat = mat.ravel()
print(flat)
```

### `.flatten()` (always copy)

```python title="flatten" showLineNumbers{1}
import numpy as np

mat = np.array([[1, 2], [3, 4]])
flat = mat.flatten()
print(flat)
```

## Add or remove dimensions

### `np.newaxis` / `None`

Convert a 1D array into a column vector:

```python title="newaxis" showLineNumbers{1}
import numpy as np

arr = np.array([1, 2, 3])
col = arr[:, None]
print(col)
print(col.shape)  # (3, 1)
```

### `np.expand_dims`

```python title="expand" showLineNumbers{1}
import numpy as np

arr = np.array([1, 2, 3])
arr2 = np.expand_dims(arr, axis=0)
print(arr2.shape)  # (1, 3)
```

### `np.squeeze` (remove size-1 dims)

```python title="squeeze" showLineNumbers{1}
import numpy as np

arr = np.array([[[1], [2], [3]]])
print(arr.shape)           # (1, 3, 1)
print(np.squeeze(arr).shape)  # (3,)
```

## Transpose (swap axes)

For 2D:

```python title="transpose" showLineNumbers{1}
import numpy as np

mat = np.array([[1, 2, 3], [4, 5, 6]])
print(mat.T)
```
