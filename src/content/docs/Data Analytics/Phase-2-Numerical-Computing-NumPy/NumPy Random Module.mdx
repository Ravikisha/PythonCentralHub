## Why random matters in analytics

Random values help with:

- Simulation and Monte Carlo experiments
- Creating synthetic datasets for testing
- Sampling
- Bootstrapping

## Recommended API: `default_rng()`

Modern NumPy recommends using a Generator:

```python title="rng" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(42)
print(rng.integers(1, 10, size=5))
```

## Seeding (reproducibility)

Seeding ensures you get the same results each run.

```python title="seed" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(123)
print(rng.normal(size=3))
```

## Random integers

```python title="integers" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(7)
arr = rng.integers(low=0, high=100, size=10)
print(arr)
```

## Random floats

```python title="random" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(7)
arr = rng.random(5)  # uniform in [0, 1)
print(arr)
```

## Normal distribution

```python title="normal" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(7)
arr = rng.normal(loc=0, scale=1, size=5)
print(arr)
```

## Choice (sampling)

```python title="choice" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(7)
categories = np.array(["A", "B", "C"])
print(rng.choice(categories, size=10, replace=True))
```

Weighted sampling:

```python title="choice-weights" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(7)
values = np.array(["low", "medium", "high"])
probs = [0.6, 0.3, 0.1]
print(rng.choice(values, size=10, p=probs))
```

## Shuffle

```python title="shuffle" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(7)
arr = np.arange(10)
rng.shuffle(arr)
print(arr)
```

## Synthetic dataset example

```python title="synthetic" showLineNumbers{1}
import numpy as np

rng = np.random.default_rng(0)

n = 100
age = rng.integers(18, 60, size=n)
income = rng.normal(loc=60000, scale=15000, size=n)

X = np.column_stack([age, income])
print(X.shape)
```
