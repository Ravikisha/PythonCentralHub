## Creating arrays from Python objects

### From a list

```python title="from-list" showLineNumbers{1}
import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr)
print(type(arr))
```

### From a nested list (2D)

```python title="from-nested" showLineNumbers{1}
matrix = np.array([
    [1, 2, 3],
    [4, 5, 6]
])
print(matrix)
print(matrix.shape)   # (2, 3)
```

### From a tuple

```python title="from-tuple" showLineNumbers{1}
arr = np.array((10, 20, 30))
print(arr)
```

## Using built-in constructors

### `np.zeros()`

Create an array filled with zeros.

```python title="zeros" showLineNumbers{1}
arr = np.zeros((2, 3))
print(arr)
```

### `np.ones()`

```python title="ones" showLineNumbers{1}
arr = np.ones((3, 2))
print(arr)
```

### `np.full()`

Create an array filled with a constant value.

```python title="full" showLineNumbers{1}
arr = np.full((2, 2), 7)
print(arr)
```

### `np.eye()` (identity matrix)

```python title="eye" showLineNumbers{1}
I = np.eye(3)
print(I)
```

### `np.arange()` (range with step)

Similar to Python `range()`, but returns a NumPy array.

```python title="arange" showLineNumbers{1}
arr = np.arange(0, 10, 2)
print(arr)  # [0 2 4 6 8]
```

### `np.linspace()` (even spacing)

Creates `num` values between start and stop.

```python title="linspace" showLineNumbers{1}
arr = np.linspace(0, 1, 5)
print(arr)  # [0.   0.25 0.5  0.75 1.  ]
```

## Specifying dtype during creation

```python title="dtype" showLineNumbers{1}
arr = np.array([1, 2, 3], dtype=np.float64)
print(arr)
print(arr.dtype)
```

## Quick recap: which function to use?

- `np.array(...)` → convert existing data (lists)
- `np.zeros(shape)` / `np.ones(shape)` → initialize arrays
- `np.full(shape, value)` → constant arrays
- `np.eye(n)` → identity matrix
- `np.arange(start, stop, step)` → integer sequences
- `np.linspace(start, stop, num)` → precise evenly spaced floats
