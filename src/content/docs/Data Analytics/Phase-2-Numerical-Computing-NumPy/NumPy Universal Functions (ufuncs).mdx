## What are ufuncs?

A **ufunc** (universal function) is a function that operates element-wise on arrays and is optimized for speed.

Examples:

- `np.sqrt`, `np.exp`, `np.log`
- `np.sin`, `np.cos`
- `np.maximum`, `np.minimum`

They are faster and cleaner than Python loops.

## Example: sqrt

```python title="sqrt" showLineNumbers{1}
import numpy as np

arr = np.array([1, 4, 9, 16])
print(np.sqrt(arr))
```

## Common math ufuncs

```python title="math" showLineNumbers{1}
import numpy as np

x = np.array([1.0, 2.0, 3.0])
print(np.exp(x))
print(np.log(x))
print(np.log10(x))
```

## Trigonometric ufuncs

```python title="trig" showLineNumbers{1}
import numpy as np

angles = np.array([0, np.pi/2, np.pi])
print(np.sin(angles))
print(np.cos(angles))
```

## Comparison ufuncs

```python title="max-min" showLineNumbers{1}
import numpy as np

a = np.array([1, 10, 3])
b = np.array([2, 5, 4])

print(np.maximum(a, b))
print(np.minimum(a, b))
```

## Working with NaN values

Some ufuncs have NaN-safe variants.

```python title="nan" showLineNumbers{1}
import numpy as np

arr = np.array([1.0, np.nan, 3.0])
print(np.nanmean(arr))
print(np.nansum(arr))
```

## Ufuncs + broadcasting

Ufuncs naturally work with broadcasting.

```python title="broadcast" showLineNumbers{1}
import numpy as np

mat = np.array([[1, 2, 3], [4, 5, 6]])
print(np.sqrt(mat))
```
