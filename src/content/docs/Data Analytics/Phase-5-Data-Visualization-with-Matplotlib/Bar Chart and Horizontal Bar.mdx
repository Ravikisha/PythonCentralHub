## Basic bar chart

```python title="Bar" showLineNumbers{1}
import matplotlib.pyplot as plt

cities = ["Pune", "Delhi", "Mumbai", "Bengaluru"]
sales = [120, 180, 90, 160]

plt.figure(figsize=(7, 4))
plt.bar(cities, sales)
plt.title("Sales by city")
plt.xlabel("City")
plt.ylabel("Sales")
plt.xticks(rotation=20)
plt.tight_layout()
plt.show()
```

## Sorted bars

```python title="Sorted bars" showLineNumbers{1}
import matplotlib.pyplot as plt

data = {"Pune": 120, "Delhi": 180, "Mumbai": 90, "Bengaluru": 160}
items = sorted(data.items(), key=lambda x: x[1], reverse=True)
labels = [k for k, _ in items]
values = [v for _, v in items]

plt.figure(figsize=(7, 4))
plt.bar(labels, values)
plt.title("Sales by city (sorted)")
plt.tight_layout()
plt.show()
```

## Horizontal bars

```python title="Horizontal bar" showLineNumbers{1}
import matplotlib.pyplot as plt

labels = ["Very long category A", "Long B", "C"]
values = [12, 19, 7]

plt.figure(figsize=(7, 4))
plt.barh(labels, values)
plt.title("Horizontal bar chart")
plt.xlabel("Value")
plt.tight_layout()
plt.show()
```

## Tip

Start y-axis at 0 for bar charts to avoid misleading comparisons.
