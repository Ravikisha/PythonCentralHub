## Matplotlib anatomy

A typical plot has:

- Title
- X label / Y label
- Axes limits
- Ticks
- Legend (if multiple series)
- Grid (optional)

## The Figure + Axes model

```python title="Figure + Axes" showLineNumbers{1}
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(7, 4))
ax.plot([1, 2, 3, 4], [10, 6, 8, 4])

ax.set_title("Anatomy")
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.grid(True, alpha=0.3)

plt.show()
```

## Limits and ticks

```python title="Limits and ticks" showLineNumbers{1}
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(7, 4))
ax.plot([1, 2, 3, 4], [10, 6, 8, 4])

ax.set_xlim(0, 5)
ax.set_ylim(0, 12)

ax.set_xticks([1, 2, 3, 4])
ax.set_yticks([0, 4, 8, 12])

ax.set_title("Limits and ticks")
plt.tight_layout()
plt.show()
```

## Tip

Use the object-oriented approach for consistent, reusable plotting.
