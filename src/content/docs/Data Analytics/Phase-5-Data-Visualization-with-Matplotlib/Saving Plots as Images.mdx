## Why saving matters

Saved plots are used in:

- Reports
- Notebooks
- Websites
- PowerPoint/Docs

## Save to PNG

```python title="savefig" showLineNumbers{1}
import matplotlib.pyplot as plt

plt.figure(figsize=(7, 4))
plt.plot([1, 2, 3], [3, 2, 5])
plt.title("Saved plot")

plt.savefig("plot.png", dpi=150, bbox_inches="tight")
```

## Notes

- `dpi` controls resolution
- `bbox_inches="tight"` avoids cutting off labels

## Save to PDF (for print)

```python title="Save PDF" showLineNumbers{1}
import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [3, 2, 5])
plt.title("PDF plot")
plt.savefig("plot.pdf", bbox_inches="tight")
```
