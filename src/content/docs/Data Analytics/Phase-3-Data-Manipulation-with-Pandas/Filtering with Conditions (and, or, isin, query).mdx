## Setup

```python title="Setup DataFrame" showLineNumbers{1}
import pandas as pd

df = pd.DataFrame({
    "name": ["Asha", "Ravi", "Meera", "Zoya", "Kabir"],
    "age": [23, 28, 26, 31, 22],
    "city": ["Pune", "Delhi", "Delhi", "Pune", "Mumbai"],
    "score": [88, 91, 77, 95, 60],
})

print(df)
```

## Basic boolean filtering

```python title="Single condition" showLineNumbers{1}
print(df[df["age"] >= 25])
```

## Multiple conditions: AND / OR

Pandas uses bitwise operators:

- AND: `&`
- OR: `|`

Always use parentheses:

```python title="AND filter" showLineNumbers{1}
mask = (df["city"] == "Delhi") & (df["score"] >= 80)
print(df[mask])
```

```python title="OR filter" showLineNumbers{1}
mask = (df["city"] == "Mumbai") | (df["score"] >= 90)
print(df[mask])
```

## `isin()` for membership checks

```python title="isin" showLineNumbers{1}
mask = df["city"].isin(["Pune", "Delhi"])
print(df[mask])
```

## `between()` for range filters

```python title="between" showLineNumbers{1}
mask = df["age"].between(23, 30, inclusive="both")
print(df[mask])
```

## `query()` for readable filters

`query()` can be easier to read, especially in notebooks.

```python title="query" showLineNumbers{1}
result = df.query('city == "Delhi" and score >= 80')
print(result)
```

### Variables inside query

```python title="query with variable" showLineNumbers{1}
min_score = 85
result = df.query('score >= @min_score')
print(result)
```

## Common mistakes

- Using `and` / `or` instead of `&` / `|` (will error)
- Forgetting parentheses around conditions
- Comparing strings with extra spaces (clean using `.str.strip()`)
