## The first 60 seconds with a new dataset

Whenever you load a new dataset, do this first.

### 1) Check shape

```python title="Shape" showLineNumbers{1}
import pandas as pd

df = pd.read_csv("data/sales.csv")
print(df.shape)  # (rows, columns)
```

### 2) Preview rows

```python title="Head / Tail" showLineNumbers{1}
print(df.head())
print(df.tail())
```

### 3) Random sample

Great for spotting weird values.

```python title="Sample rows" showLineNumbers{1}
print(df.sample(5, random_state=42))
```

### 4) Columns and dtypes

```python title="Info" showLineNumbers{1}
df.info()
```

This tells you:

- Column names
- Non-null counts
- Dtypes
- Memory usage (helpful when data grows)

### 5) Descriptive stats

```python title="Describe" showLineNumbers{1}
print(df.describe())
```

For categorical columns:

```python title="Describe object columns" showLineNumbers{1}
print(df.describe(include=["object"]))
```

## Core sanity checks

### Missing values per column

```python title="Missing values" showLineNumbers{1}
missing = df.isna().sum().sort_values(ascending=False)
print(missing)
```

### Duplicates

```python title="Duplicate rows" showLineNumbers{1}
print("duplicate rows:", df.duplicated().sum())
```

### Value counts for a category

```python title="Value counts" showLineNumbers{1}
print(df["city"].value_counts(dropna=False).head(10))
```

## Tip: create an inspection helper

```python title="Quick inspection helper" showLineNumbers{1}
import pandas as pd

def inspect(df: pd.DataFrame, n: int = 5) -> None:
    print("shape:", df.shape)
    print("columns:", list(df.columns))
    print("\nhead:")
    print(df.head(n))
    print("\nmissing:")
    print(df.isna().sum())

# inspect(df)
```
