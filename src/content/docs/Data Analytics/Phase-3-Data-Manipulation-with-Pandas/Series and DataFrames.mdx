## Creating a Series

A **Series** is a labeled 1D array.

```python title="Create a Series" showLineNumbers{1}
import pandas as pd

s = pd.Series([10, 20, 30], name="scores")
print(s)
```

### With a custom index

```python title="Series with explicit index" showLineNumbers{1}
import pandas as pd

s = pd.Series([10, 20, 30], index=["a", "b", "c"], name="scores")
print(s)
print("Index:", s.index)
```

## Creating a DataFrame

A **DataFrame** is a labeled 2D table.

### From a dictionary of columns

```python title="DataFrame from dict" showLineNumbers{1}
import pandas as pd

df = pd.DataFrame({
    "name": ["Asha", "Ravi", "Meera"],
    "age": [23, 28, 26],
    "city": ["Pune", "Delhi", "Bengaluru"],
})

print(df)
```

### From a list of dictionaries (records)

```python title="DataFrame from records" showLineNumbers{1}
import pandas as pd

rows = [
    {"name": "Asha", "age": 23, "city": "Pune"},
    {"name": "Ravi", "age": 28, "city": "Delhi"},
]

df = pd.DataFrame(rows)
print(df)
```

## Understanding index and columns

- `df.index` labels rows
- `df.columns` labels columns

```python title="Index and columns" showLineNumbers{1}
import pandas as pd

df = pd.DataFrame({"a": [1, 2], "b": [3, 4]})
print("index:", df.index)
print("columns:", df.columns)
```

## Dtypes: why they matter

Pandas stores each column using a data type (`dtype`).

```python title="Check dtypes" showLineNumbers{1}
import pandas as pd

df = pd.DataFrame({
    "age": [20, 21, 22],
    "score": [90.5, 88.0, 91.0],
    "passed": [True, True, False],
})

print(df.dtypes)
```

### Common dtype pitfalls

- A numeric column can become `object` if it contains mixed types like `"N/A"`.
- Dates start as strings unless you parse them.
