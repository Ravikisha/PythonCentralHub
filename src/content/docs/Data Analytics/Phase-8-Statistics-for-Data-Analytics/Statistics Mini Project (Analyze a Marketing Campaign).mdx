## Goal

Given a marketing campaign dataset, you will:

- compute descriptive statistics
- compare two segments (A vs B)
- build confidence intervals
- produce a short written conclusion

## Example dataset columns

- `user_id`
- `variant` (A/B)
- `converted` (0/1)
- `revenue`
- `country`

## Step 1: Load

```python title="Load campaign data" showLineNumbers{1}
import pandas as pd

df = pd.read_csv("data/campaign.csv")
print(df.head())
```

## Step 2: Quick summary

```python title="Group summary" showLineNumbers{1}
summary = (
    df.groupby("variant")
      .agg(
          users=("user_id", "nunique"),
          conversion_rate=("converted", "mean"),
          avg_revenue=("revenue", "mean"),
          median_revenue=("revenue", "median"),
      )
)
print(summary)
```
