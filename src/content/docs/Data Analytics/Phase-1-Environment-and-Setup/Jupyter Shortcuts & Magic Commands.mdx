## Why shortcuts and magics matter

In data analytics, you often:

- Rerun code many times
- Explore different plots and transformations
- Measure performance
- Run shell commands

Jupyter shortcuts and magic commands help you iterate much faster.

## Keyboard shortcuts (classic Jupyter Notebook)

Jupyter has two main modes:

- **Command mode** (press `Esc`): navigate and manage cells
- **Edit mode** (press `Enter`): edit inside a cell

### Most useful shortcuts (Command mode)

- `A` Ôćĺ insert cell **above**
- `B` Ôćĺ insert cell **below**
- `D, D` Ôćĺ delete selected cell
- `X` Ôćĺ cut cell
- `C` Ôćĺ copy cell
- `V` Ôćĺ paste cell
- `Z` Ôćĺ undo delete
- `M` Ôćĺ convert cell to **Markdown**
- `Y` Ôćĺ convert cell to **Code**
- `Shift + Up/Down` Ôćĺ select multiple cells

### Most useful shortcuts (Edit mode)

- `Ctrl + Enter` Ôćĺ run cell (stay)
- `Shift + Enter` Ôćĺ run cell (move down)
- `Alt + Enter` Ôćĺ run cell (insert new cell below)
- `Tab` Ôćĺ autocomplete (when available)
- `Ctrl + /` Ôćĺ toggle comment in selection

:::tip
Shortcut cheat sheet
In classic notebook: `Help` Ôćĺ `Keyboard Shortcuts`.
:::

## What are magic commands?

**Magic commands** are special Jupyter commands that start with:

- `%` for **line magics**
- `%%` for **cell magics**

They are not standard Python syntax; theyÔÇÖre handled by IPython/Jupyter.

## Common line magics

### `%time` and `%timeit`

Use these to measure execution time.

```python title="time" showLineNumbers{1}
%time sum(range(10_000_00))
```

`%timeit` runs the expression multiple times and gives an average:

```python title="timeit" showLineNumbers{1}
%timeit sum(range(1_000_00))
```

### `%who` and `%whos`

Show variables currently defined.

```python title="who" showLineNumbers{1}
%whos
```

### `%pwd` and `%cd`

- `%pwd` shows current working directory
- `%cd` changes it

```python title="pwd" showLineNumbers{1}
%pwd
```

```python title="cd" showLineNumbers{1}
%cd data
```

### `%ls` (list files)

```python title="ls" showLineNumbers{1}
%ls
```

## Plotting magics

### `%matplotlib inline`

Classic way to render plots inside the notebook.

```python title="matplotlib-inline" showLineNumbers{1}
%matplotlib inline
```

### `%matplotlib notebook` (interactive)

This can enable interactive plots in the classic notebook.

```python title="matplotlib-notebook" showLineNumbers{1}
%matplotlib notebook
```

:::note
In newer setups, Matplotlib interactivity depends on your environment and extensions.
If `%matplotlib notebook` doesnÔÇÖt work, you can still use inline plots.
:::

## Useful cell magics

### `%%writefile` (save code to a file)

You can write a cellÔÇÖs content into a Python script.

```python title="writefile" showLineNumbers{1}
%%writefile my_script.py
print("Hello from a file")
```

### `%%bash` (run bash commands)

If your Jupyter supports it, you can run bash scripts:

```bash title="bash" showLineNumbers{1}
%%bash
ls -la
```

If `%%bash` isnÔÇÖt available, you can use `!` shell escapes instead.

## Shell escapes with `!`

Prefix a command with `!` to run it in the OS shell.

```python title="shell-escape" showLineNumbers{1}
!python --version
```

```python title="pip-list" showLineNumbers{1}
!pip list | head
```

## Getting help quickly

### Add `?` to functions

```python title="help" showLineNumbers{1}
import pandas as pd
pd.DataFrame?
```

### Use `help()`

```python title="help2" showLineNumbers{1}
help(pd.DataFrame)
```

## Common mistakes

### Mistake 1: Using magics in `.py` files

Magics only work in Jupyter/IPython, not in normal Python scripts.

### Mistake 2: Forgetting to rerun earlier cells

Notebook state is incremental. If you restart the kernel, rerun setup cells.
